# MusterFlow Community Catalog

Turn any API into a CLI, MCP tool, and workflow. This repository is the community catalog — submit OpenAPI specs to share tools with everyone.

## How to Contribute

1. Add your API's OpenAPI spec and metadata to `entries.json`
2. Open a PR
3. CI validates the spec and quality-scores it
4. Merged → live on [catalog.musterflow.com](https://catalog.musterflow.com)

Or use the CLI:
```
musterflow catalog push
```

## Quality Tiers

- **Official** — Published by the API provider, domain-verified
- **Community** — Validated OpenAPI spec, community-submitted
- **Untested** — Inferred or unverified spec, use at own risk
